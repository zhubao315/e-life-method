## 这是什么

e-Life 是一套完整的个人知识管理系统，包含：

- **4 大理论基石**：知识复利、四维身份、转化阶梯、人机共生
- **4 大系统方法**：空间架构、双轴标签、MOC 导航、项目驱动
- **完整实践指南**：日常节奏、笔记规范、质量治理、人机协作

## 核心理念

```
知识的价值不在于拥有多少，而在于连接多少。连接产生复利。
```

```
原始输入 → 概念 → 模型 → 框架 → 原则 → 洞察
(是什么)   (定义)  (结构)  (行动)  (准则)  (悟道)
```
